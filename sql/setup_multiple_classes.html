הוספת תמיכה במספר כיתות למורה

כדי לאפשר למורה להיות אחראי על יותר מכיתה אחת, עלינו להוסיף עמודה חדשה בבסיס הנתונים שתכיל רשימה (מערך) של מזהי כיתות.

שלב 1: הרצת הסקריפט ב-Supabase

העתק את הקוד הבא והרץ אותו ב-SQL Editor של פרויקט ה-Supabase שלך:

-- 1. Add class_ids array column to profiles table
ALTER TABLE public.profiles 
ADD COLUMN IF NOT EXISTS class_ids UUID[] DEFAULT '{}';

-- 2. (Optional) Migrate existing class_id to the new array for existing users
UPDATE public.profiles 
SET class_ids = ARRAY[class_id]
WHERE class_id IS NOT NULL 
AND (class_ids IS NULL OR cardinality(class_ids) = 0);

-- 3. Update the add_existing_user_to_campaign function to handle class_ids if needed
-- (The UI handles the update, but this ensures the column is queryable)
שים לב: לאחר הרצת הקוד, הממשק יתעדכן אוטומטית ויאפשר לך לבחור מספר כיתות בעת יצירת או עריכת מורה.

שלב 2: רענון האפליקציה

לאחר הרצת הסקריפט, רענן את דף הניהול באפליקציה. כעת תוכל לראות רשימת כפתורים לבחירה מרובה של כיתות למורים.
